# Projek Burido Saleh

Ini adalah dokumentasi proyek "buridosaleh" menggunakan React Native Expo.

## Persiapan

Pastikan Anda memiliki Node.js dan Expo CLI terinstal di mesin Anda sebelum memulai.

## Langkah-langkah

### 1. Membuat Proyek Baru

Buka terminal dan jalankan perintah berikut untuk membuat proyek baru dengan Expo CLI:

```cmd
yarn android
```
